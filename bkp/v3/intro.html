Instrução para Uso do Slide

Utilize os seguintes comandos para ter melhor proveito do slide:

  • Navegação: Clique nas setas da esquerda ou direita para navegar pelos slides, ou aperte a seta esquerda ou direta do teclado.
  • Pular slide: Para pular diretamente para outro slide, aperte a tecla "G" e digite o número do slide, aperte enter para pular para o slide selecionado ou fechar o formulário.
  • Índice: Para visualizar os slides, aperta a tecla "M" para acionar o indice e navegar até o slide desejado, clique "M" para voltar aos estudos.
